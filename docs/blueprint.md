# **App Name**: RandomTalk Online

## Core Features:

- Random Voice Chat: Connect users in real-time, one-on-one voice chats with random strangers.
- Call History: Keep a call history for a limited time with users you interacted with.
- Add Friend: Ability to add a connected user as a friend.
- User Reporting: Allow users to report inappropriate behavior during chats.
- Auto Call: Enable Auto Call to call automatically another random user once a call is completed.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to evoke a sense of connection and technology.
- Background color: A dark navy blue (#1A237E), desaturated and darkened to 20% saturation and 20% brightness, to ensure readability of text and other visual elements.
- Accent color: A bright green (#90EE90), approximately 30 degrees to the 'right' of the primary hue, used for highlighting interactive elements and status indicators.
- Body and headline font: 'Inter', a sans-serif font providing a modern and neutral feel, which supports readability.
- Simple, outlined icons to represent actions like mute, hang up, and report.
- A clean, centered layout with clear call-to-action buttons for starting and ending chats.
- Smooth transitions and subtle animations when connecting and disconnecting calls.
